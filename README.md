# Quran App

A web-based application for reading and interacting with the Holy Quran.

## Features
- Digital Mushaf experience
- Custom fonts for authentic Quranic script
- Hifz data management
- Responsive design

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- Python (for data processing)

## Getting Started
Simply open `index.html` in any modern web browser.
